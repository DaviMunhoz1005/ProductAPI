# Product_API

The project is a product control api, it was built entirely with the spring framework and several of its dependencies, the api has unit and integration tests, security configuration in requests and also has "custom" exceptions.

## Used databases:

MySQL and H2(for testing)

## Requests

### For USERS:

- Find all products in pageable

- Find all products

- Find one
product by id

- Add a product

- Replace a product
Find a product by name

### For ADMIN:

- Find a product by id and user details

- Delete a product
